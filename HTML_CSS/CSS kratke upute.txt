Preporuke za definiranje stila fonta
------------------------------

Velièinu slova možete odrediti u pixelima ili em.
Preporuka: koristiti em.
Zašto? Da bi korisnik mogao poveæati font u izborniku browsera. To je i preporuka W3C.

Kako preraèunati iz pixela u em jedinice?
1em is equal to the current font size. The default text size in browsers is 16px. So, the default size of 1em is 16px.
The size can be calculated from pixels to em using this formula: pixels/16=em

Primjer:

body {
    background-color: light-gray;
    font-size: 100%;
}

h1 {
    font-family: Arial;
    color: black;
    text-align: center;
    font-size: 2.5em; /* adekvatno 40px */
}
